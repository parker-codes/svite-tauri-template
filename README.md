# svite-tauri-template

To clone this, run `npx degit parker-codes/svite-tauri-template`.

For Tauri setup, visit [tauri.studio](https://tauri.studio/).

```
# in one tab
yarn dev

# in another
yarn tauri dev
```

### Components

- svelte (UI)
- svite (bundler plugin for vite)
- tailwind (styling)
- tauri (app)
